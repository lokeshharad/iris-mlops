VERSION: 1.4.0.0

VERSION FORMAT: MAJOR.MINOR.PATCH.BUILD

=== VERSION COMPONENT DESCRIPTIONS ===

MAJOR (1):
  - Represents major release changes or architecture shifts
  - Increment when the project structure or deployment model changes significantly

MINOR (4):
  - Represents new features or workflow enhancements
  - Increment when adding major capabilities such as registry-based deployment

PATCH (0):
  - Represents bug fixes and small refinements

BUILD (0):
  - Represents the current internal build iteration

=== VERSION HISTORY ===

v1.4.0.0 - Current Release
  Features & updates:
    - Training now logs and registers the selected model in the MLflow Model Registry
    - Registry names are dynamic and based on the winning model and metric value
    - Prediction loads models from the MLflow registry instead of relying on local model files
    - Local models-folder artifact flow was removed from the prediction path
    - README updated to reflect the registry-based workflow

v1.3.0.1 - Previous Release
  Features & updates:
    - Refactored training and prediction into package-based modules
    - Added CLI-based training and prediction entrypoints
    - Improved CSV handling and evaluation logic

v1.2.0.3 - Previous Release
  Features & updates:
    - Added model versioning with timestamped local model files
    - Added latest pointer management for local model artifacts
